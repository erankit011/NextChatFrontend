# NextChat Frontend

Frontend application for NextChat real-time chat.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Socket.IO Client
- React Router

## Folder Structure

```
client/
├── pages/            # Page components
├── src/
│   ├── components/   # Reusable components
│   ├── services/     # API services
│   ├── context/      # React context
│   ├── hooks/        # Custom hooks
│   └── utils/        # Utility functions
└── public/           # Static assets
```

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
